# translation_tools
#### 一个基于有道翻译以及百度翻译API开发的翻译小工具
#### 使用的技术
- VUE
- Element-ui
- 有道智云API
- 百度翻译API

#### 使用说明
申请有道智云API或者百度翻译API得到应用ID以及应用密钥并填入输入框即可！
